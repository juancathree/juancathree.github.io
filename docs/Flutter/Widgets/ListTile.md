# List tile

```dart
 ListTile(
  leading: Icon(Icons.star), // Icono a la izquierda del título
  title: Text('Simple ListTile'), // Título principal
  subtitle: Text('Subtitle'), // Subtítulo opcional
  trailing: Icon(Icons.arrow_forward), // Icono a la derecha del título
  onTap: () {
    // Acción al hacer clic en el ListTile
    print('ListTile tapped!');
  },
),
```
