# Sized Box
```dart
SizedBox(
  width: 200.0, // Ancho del cuadro
  height: 100.0, // Altura del cuadro
  child: Container(
    color: Colors.blue,
    child: Center(
      child: Text(
        'Hello, SizedBox!',
        style: TextStyle(color: Colors.white),
      ),
    ),
  ),
)
```
